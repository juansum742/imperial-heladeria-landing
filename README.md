# Imperial Heladería - React landing demo

Landing page profesional en React generada desde el artefacto Open Design original.

## Comandos

```bash
npm install
npm run dev
npm run build
npm run preview
```

El build de producción se genera en `docs/` para publicarlo con GitHub Pages desde la rama `main`.
El HTML original de Open Design permanece disponible como `imperial-heladeria-landing.html`.
